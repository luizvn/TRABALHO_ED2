SOBRE

Nossa História

A Royal Sports foi criada com o propósito de oferecer produtos esportivos de alta qualidade, combinando inovação e excelência para atender atletas de todos os níveis. Fundada por apaixonados pelo esporte, nossa missão é proporcionar uma experiência diferenciada, garantindo que cada item vendido seja sinônimo de desempenho e confiança.

Trabalho

Este site faz parte de um trabalho acadêmico desenvolvido para a disciplina de Estrutura de Dados II, ministrada pela doscente Maria Inês. O projeto tem como objetivo aplicar conceitos avançados de estruturas de dados na implementação de um e-commerce de produtos esportivos.

O funcionamento do site está dividido em três etapas principais:

  • Catálogo: CRUD completo para gerenciamento dos produtos esportivos, permitindo adicionar, editar e excluir itens.
  • Busca: Implementação de uma busca eficiente utilizando Árvores B para otimizar a recuperação de produtos.
  • Compra: Funcionalidade de compra, utilizando o mapa Leaflet e grafos para calcular rotas de entrega, estimar o frete e gerenciar o carrinho de compras.
